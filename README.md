# TikoPlay (Tauri + Python)

Cross-platform TikTok Live controller.
UI: Tauri (HTML/CSS/JS)
Backend: Python (TikTokLive + pyautogui)

## Development
- Install Node.js 18+
- Install Rust
- Install Python 3.10+

## Run UI
npm install
npm run tauri dev

## Run listener standalone
python python/listener.py
